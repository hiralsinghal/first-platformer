# First Platformer

This is my first game that I made in Godot and worked on myself, seeing tutorials and implementing stuff.

The map asset was created by me. I messed up somewhat but it works.

The game ends when you reach the end of the map. I wasn't able to implement a loading screen.

I created this because I always wanted to explore making a game and learn something new as well. I enjoyed making it.

## How to play?

```A or Left arrow key``` - Move left

```D or Right arrow key``` - Move right

```W or Up arrow key``` - Jump
